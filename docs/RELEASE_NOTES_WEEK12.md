# Release Notes - Week12

## Summary
Week12では公開導線の安定化と収益導線の強化を同時に進め、運用センターでの意思決定速度を上げる改善を実装しました。

## Implemented
- 作成タブに「規模別の最短テンプレ提案」と「QR配布後24hアクション」を追加
- LPに季節要因を加味した業態別訴求を追加
- LP事例カードに季節・業態優先の並び替えロジックを追加
- 編集画面の公開後チェックを自動採点化（スコア表示）
- 問い合わせ導線のクイック補完を重複防止付きで強化
- 画像最適化候補向けの「一括変換ガイド」コピー導線を追加
- `/api/ops/health` に `week12Preview` を追加
- 運用センターにWeek12 KPIレビュー（再公開率/紹介流入/デバイス×流入CTA率など）を追加
- 請求導線の離脱率可視化と阻害要因別アクション提案を追加
- 週次レポートに運用負荷削減時間と紹介流入率を追加
- 優先カード順をKPI連動で自動表示

## Ops Notes
- 重大通知ルートは `OPS_ALERT_SLACK_WEBHOOK_URL` と `OPS_ALERT_EMAIL_TO` の有無で自動判定
- 既存ログから復旧中央値（分）を推定表示
