# Release Notes - Week7

## 概要
Week7では、ホテル向けのLP最適化・初回公開導線・運用センター分析を強化し、CVR改善の実行速度を高めました。

## 主な変更
- 業態別LP（business/resort/spa）でヒーロー訴求と下部CTAを差し替え
- LP別の勝ちCTAパターンを運用センターで自動表示
- 新規登録後の初回公開ウィザードを3画面化し、離脱/完了ログを計測
- 1画面目完了率をダッシュボードに追加
- 施設タイプ別の固定Top3テンプレを作成タブに追加
- 低利用テンプレへ「改善候補」タグを追加
- テンプレ選択→公開までの中央値を運用センターに表示
- 公開前チェックでエラー種別の発生率と推奨修正を可視化
- 速度改善提案をLCP/INP/CLSベースで自動表示
- 画像最適化の一括変換オペレーション（URLリスト出力）を追加
- 優先度スコアに改善工数（S/M/L）を追加
- 休眠通知（3/7/14日）の配信ログ記録を追加
- 再開施策の勝ち導線固定（記録）を追加
- 週次レポートに「今週の最優先3施策」を追加
- Week7 KPIレビューカードを運用センターに追加

## 技術ノート
