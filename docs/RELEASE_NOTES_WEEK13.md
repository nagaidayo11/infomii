# Release Notes - Week13

## Summary
Week13では、収益化導線と継続運用導線を同時に強化し、LP→登録→公開→課金→再開の一連フローを運用センターで追える状態にしました。

## Implemented
- LP
  - 検索流入時に検索ユーザー向けFAQを優先表示
  - CTA文言を「30秒登録」軸で統一
  - 事例セクションを「課題→解決→効果」構成に変更
  - 成果数値カードを固定表示
  - プレビュー画像URL最適化（主要CDN向けパラメータ付与）
- Login / Create
  - ログイン前に業態選択を追加し、作成画面の初期テンプレ推薦へ引き継ぎ
- Editor
  - ホテル向け完成テンプレパックをクイック適用
  - 公開前チェックでhttpリンクを危険項目としてエラー化
  - 画像圧縮ガイドに実行チェックを追加
- Ops Center / Dashboard
  - `week13Preview` 追加
  - 課金完了率の日次7日可視化
  - 通知後14日再公開率（チャネル別）可視化
  - 今週の最優先3施策を自動生成
  - Webhook再送チェックガイドを追加
  - `past_due` の自動リマインド推奨間隔を表示
  - 週次レポート自動送信テンプレのコピー導線を追加
  - Week13 KPIレビューカードを追加

## Notes
- Webhook再送はStripeダッシュボードでの操作が必要です。
- 重大通知のSlack/Mail有効化は環境変数設定に依存します。
