# Week8 Execution Tickets (Conversion Expansion)

## Goal
ホテル特化のCVR改善を維持したまま、紹介獲得と継続課金率を拡張する。

## Day1
- [ ] 業態別LPの勝ち訴求を固定化し、負け訴求を停止
- [ ] LPのCTA文言を勝ちパターンのみ表示するモードを追加
- [ ] LP遷移元SNSごとの最適バリアントを自動適用

## Day2
- [ ] 初回公開ウィザードに「QR配布」完了チェックを追加
- [ ] ウィザード離脱時に再開リンクを自動表示
- [ ] ウィザード完了ユーザーの7日継続率を可視化

## Day3
- [ ] 施設タイプ別テンプレTop3を固定し、月次で再評価
- [ ] 低利用テンプレの改善案を自動生成
- [ ] テンプレ選択→公開中央値を業態別で比較表示

## Day4
- [ ] 公開前チェックのエラー修正をバッチ適用
- [ ] よくある修正をワンクリックテンプレ化
- [ ] 公開前チェックスコア80未満のブロックを自動ハイライト

## Day5
- [ ] 画像最適化の半自動変換フローを追加
- [ ] 速度改善施策の実行前後比較を表示
- [ ] 速度優先度（S/M/L）を作業時間見積もり付きで表示

## Day6
- [ ] 休眠通知の配信結果（既読/未反応）を記録
- [ ] 再開勝ち導線を施設タイプ別にプリセット化
- [ ] 週次レポートを管理者向けダッシュボードカードに自動掲示

## Day7
- [ ] Week8 KPIレビュー（LP CVR / 公開率 / Pro転換 / 継続）
- [ ] Week9チケットを作成
- [ ] Week8リリースノートを作成
