# UI 表記ルール

- **対象ユーザー**: 日本のホテル現場スタッフ
- **UI 文言**: 原則すべて**日本語**（ボタン・見出し・表ヘッダー・プレースホルダー・トースト等）
- **商品名**: 画面内では「インフォミー」表記を推奨（ロゴ・メタデータの英字表記は別途可）
- **コード上の識別子**（`type: "text"` 等）や **URL スラッグ**は英語のままでよい

新規コンポーネントを追加するときは、英語のまま置かないこと。

## 統一ラベル（ボタン・主要アクション）

| ラベル | 用途の目安 |
|--------|------------|
| **ページ作成** | 新規ページ・テンプレ適用・作成導線の主ボタン |
| **保存** | 下書きの保存（JSONダウンロードやAPI保存など） |
| **プレビュー** | ゲスト表示の確認パネル見出し |
| **QRコード生成** | QR画像を開く・取得するアクション |
| **編集** | エディタへ入る・施設名などを直す |
| **削除** | ブロックやページの削除 |
| **公開** | 公開状態にする・公開導線 |
